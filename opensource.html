How to Contribute

If you have better handwritten notes, newer past papers, or want to add features (like a dark mode or search bar) to this website, contributions are highly encouraged.

  1. Fork the repository on GitHub.
  2. Commit your updated notes or code changes.
  3. Submit a Pull Request for review.
