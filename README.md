# 📚 MLSU LLB Study Resources

A student-driven, open-source initiative to help law students at Mohan Lal Sukhadia University (MLSU) prepare and succeed together. 

## 🎯 Overview
This repository contains the source code for the MLSU Study Resources website, alongside organized archives of handwritten notes and previous year examination papers. 

### 📂 What's Included
* **Complete LLB Study Materials:** Focused resources to help you pass your exams.
* **All 11 Papers:** Includes materials for all core subjects.
* **Specialized Subjects:** Features hard-to-find notes for **Legal Language** and **General Hindi**.
* **Handwritten Notes:** Created by students, for students, to make quick revision easier.

## 🚀 Live Website
You can access and download the materials directly from the live website here: 
**[https://mlsu-dhj.pages.dev/]**
